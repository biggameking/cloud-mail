## 安全审计

- 普通用户不能访问 AI 管理 API；所有业务入口再次校验超级管理员身份。
- 18 个 AI API 的未认证集成测试均返回 401；所有 AI 服务边界在普通用户身份下于访问数据库前返回 403；全部 AI 写接口和预览接口均有独立速率限制。生产 `/api/ai/metrics` 未登录请求也返回 401。
- 摘要流筛选只返回摘要、运行元数据和计数；邮箱筛选通过来源关系做存在性检查，不把源邮件主题、正文或 HTML加入列表响应。
- 模型无工具、无附件、无数据库或外发权限，只接受清洗后的文本并返回受校验 JSON。
- 环境、数据库主开关、投递开关三层独立；投递关闭时禁止新投递及历史重试。
- 自动投递增加规则级第四层开关；规则未明确选择、被关闭或软删除时，不生成待投递摘要，也不重试历史待投递记录。
- 历史摘要来源在每次查看时重新检查源邮件、邮箱及所属用户仍有效；失效来源不会返回。
- 显式投递只读取 Cloudflare Secret `AI_DIGEST_DESTINATION_SECRET` 中的目标地址，请求体不能指定收件人；部署脚本还用本机忽略配置把 Send Email binding 限制到同一目标，临时配置部署后立即删除；每摘要最多 3 次尝试并受速率限制。
- `.env.local` 已被 Git 忽略；当前跟踪文件未发现私钥、API Key 或 Token 值。
